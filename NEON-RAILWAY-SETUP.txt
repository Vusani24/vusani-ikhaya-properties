Alexandra Rooms To Rent - Neon + Railway Setup

Neon is the recommended free database option if Supabase is not available for you.

1. Go to https://neon.com
2. Sign up or log in.
3. Create a new project.
4. Project name:
   Alexandra Rooms To Rent
5. Choose the Free plan.
6. After the project opens, copy the connection string. It starts with:
   postgresql://

7. In Neon, open SQL Editor.
8. Open this file from the project:
   database/neon-schema.sql
9. Copy all SQL and run it in Neon.

10. In Railway, open your Alexandra website service.
11. Go to Variables.
12. Add:

   ADMIN_PASSWORD=Av98012@12
   DATABASE_URL=your Neon connection string

13. Remove Supabase variables if you are not using Supabase:

   SUPABASE_URL
   SUPABASE_SERVICE_ROLE_KEY

14. Redeploy the Railway service.

15. Test:

   https://your-domain/api/status

It should show:

   "mode": "postgres-database-url"
   "ok": true

Important:
- Do not paste DATABASE_URL inside public HTML or JavaScript.
- DATABASE_URL belongs only in Railway Variables.
- The first time the server connects to Neon, it can copy the current database/db.json into Neon if the app_state table is empty.
